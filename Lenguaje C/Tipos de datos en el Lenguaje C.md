 
En el [[Lenguaje C]] existen 5 tipos de datos
| Tipo de dato | Descripción                            |
| ------------ | -------------------------------------- |
| [[char]]     | Carácter o entero pequeño (byte)       |
| [[int]]      | Entero                                 |
| [[float]]    | Punto flotante                         |
| [[double]]   | Punto flotante (mayor rango que float) |
| [[void]]     | Sin tipo (uso especial)                |


**Ejemplos:**
```c
float a;
int b, c;
char m, n;
```
